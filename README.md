# Humandeck Web MVP

바이럴 테스트 실험용 웹 MVP입니다. 핵심 목표는 아래 3개입니다.

- 완주율
- 공유율
- 광고 노출/체류 시간

## 포함한 흐름

- 랜딩 카피와 시작 CTA
- 12문항 3지선다 테스트
- 결과 직전 필수 광고 슬롯
- 애착 유형 1개 + 도파민 유형 1개 결과 공개
- 광고 시청 후 상세 해설 열기
- 공유용 결과 문구 복사/웹 공유

## 실행

```bash
npm install
npm run dev
```

## 광고/분석 다음 단계

- 결과 직전 광고 슬롯에 실제 웹 광고 SDK 연결
- `광고 보고 상세 열기` 버튼에 보상형 광고 완료 이벤트 연결
- 공유 버튼에 카카오 SDK 또는 자체 공유 링크 연결
